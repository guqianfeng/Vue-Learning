# 01-课程介绍 

## 知识大纲

### 为什么学习Vue
* Github趋势来看，Vue目前排在第一
* 基于HTML的语法，响应式的更新机制，可以更快更高效的开发项目
* 渐进式的理念，繁华的生态圈，为我们提供了大量的最佳实践

### 课程特色
* 相对完整构建Vue各种知识点
* 基础 => 生态 => 实战
* 基础
    * 属性
    * 事件
    * 插槽
    * 指令
    * 生命周期
* 生态
    * 路由管理器-Vue Router
    * 状态管理器-Vuex
    * 同构-Nuxt 
* 实战
    * 大型开源项目 - Ant Design Pro
    * 从0开始构建
    * 开发-测试-打包-构建-上线
