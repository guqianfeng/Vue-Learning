# 如何发布组件到npm

## 知识大纲

* npm需要注册账号
* package.json最少需要2个字段，name和version
  * name是全局唯一的，如果别人发过相同的名称，就不能使用
  * description - 描述
  * keywords - 关键词
  * main - 指定我们的入口文件
  * files - 指定我们哪些文件要发在npm上
* 不能使用淘宝的源，需要使用官方的源
* 推荐使用nrm，可以方便切换源
